# 单例消息

Ant Design Vue 中所提供的`message`多例的，在使用过程中用户体验不是很好，计划构建一个单例消息组件，包含如下属性：

1. 消息类型（type: info、error、warn、success）
2. 消息文本（message）
3. 持续时间（durnation，带有倒计时功能）
4. 关闭倒计时（closeTime）
4. 绑定对象（value，方便关闭回调通过特定值执行）
5. 关闭回调函数
