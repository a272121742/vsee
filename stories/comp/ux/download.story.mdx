# 下载器

Ant Design 下没有下载按钮，很多地方下载的时候比较麻烦，有的地方是直接点击下载链接，有的地方是通过接口获取下载地址再进行下载。并且没有进度显示，也没有回调，用户体验非常的不好。预计增加一个下载器（包含按钮和链接），通过原生`xhr`的方式实现下载，并提供：

1. 适配两种下载模式（下载链接和接口调用，如果是接口调用，需要后端返回给我`Content-Length`才能显示进度）；
2. 可以显示下载进度，通过背景颜色来显示进度，也是通过颜色来区分是否下载成功；
3. 下载时禁止重复点击；
4. 下载后触发事件通知，外部可以调用此事件对用户进行相关通知，例如弹窗或者其他操作；
