## 参考

```css
@font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, 'Noto Sans', sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol', 'Noto Color Emoji';
```

这是一套比较完整的字体方案集，也是`Ant Design`中默认制定的。

## 定制

如果希望使用额外的字体，就需要进行额外的添加。为了进行兼容性强的兼容，我们强烈建议使用`@font-face`方案，例如，这里使用了**思源黑体**和**roboto**：

```css
/* 定义思源黑体字体集 */
@font-face {
  font-family: "SYHT";
  src:url("/static/font/SourceHanSansCN-Regular.woff2") format("woff2"),
      url("/static/font/SourceHanSansCN-Regular.woff") format("woff"),
      url("/static/font/SourceHanSansCN-Regular.ttf") format("truetype"),
      url("/static/font/SourceHanSansCN-Regular.eot?#iefix") format("embedded-opentype"),
      url("/static/font/SourceHanSansCN-Regular.svg") format("svg");
  font-weight: normal;
  font-style: normal;
}
/* 定义roboto字体集 */
@font-face {
  font-family: "roboto";
  src:url("/static/font/Roboto-Regular.woff2") format("woff2"),
      url("/static/font/Roboto-Regular.woff") format("woff"),
      url("/static/font/Roboto-Regular.ttf") format("truetype"),
      url("/static/font/Roboto-Regular.eot?#iefix") format("embedded-opentype"),
      url("/static/font/Roboto-Regular.svg") format("svg");
  font-weight: normal;
  font-style: normal;
}
*, :root, html, body, #app {
  font-family: "SYHT", "roboto", -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, 'Noto Sans', sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol', 'Noto Color Emoji'!important;
}
```

> 注意：
>
> 1. 定义的字体都在静态文件夹`/static/font`中；
> 2. 重新覆盖原生的字体集；
> 3. 中文字体集由于较大，所以在选择时慎重，除非用户需求；
> 4. 即便选择中文字体集，也应该只加载中文字体集，有的字体还会包含韩文、日文等亚洲字体集；
