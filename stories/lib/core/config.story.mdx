# 配置中心

每个独立的项目中，都有一个`config.js`文件，这个文件非常的关键，下面公布这个文件中的所有配置：

以下为`vsee@1.2.x`的配置。在`1.2.x`版本中，我们对配置中心进行了大量的重构，以下是全部配置（已公开配置为可用配置，其他未公开配置请联系作者）：
