# 编辑器环境

`vsee`平台为了方便开发者进行开发，需要`vscode`相关配置与其联合，包含：

1. `vue`开发工具套件；
2. `eslint`检测与修复；

为此我们将一份简易的配置进行了公开，我们将会使用`vscode`中额外的插件`Settings Sync`进行配置。

## 同步配置

首先，我们需要在`vscode`中安装插件`Settings Sync`，可自行搜索，或者[点击这里进行下载](https://marketplace.visualstudio.com/items?itemName=Shan.code-settings-sync)。

下载完后，打开`vscode`命令面板，你可以有两种方式打开：

1. 打开工具栏 -> 查看 -> 命令面板；
2. 使用快捷键，Mac（shift + commond + p），Win（shift + ctrl + p）；

接着输入`sync`，命令面板会提示和`sync`有关的命令，选择“Sync：下载配置（Download Settings）”，此时会弹出如下界面：

<img src={SettingsSyncPng} />

点击右下角的"Download Public Gist", 此时命令面板会弹出输入`Gist`的输入框，输入如下代码：

> `d0af5d31355d1a49b6f02efbb7e9f6b1`

然后等待`vscode`下载完所有的配置和扩展程序即可。致此，你将和所有开发过`vsee`的开发者一样拥有相同的开发环境。当然，你只能下载配置，而不能上传配置，但你可以更改自己本地的配置作为自己开发时的**瑞士军刀**。
