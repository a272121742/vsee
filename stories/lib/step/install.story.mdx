# 安装须知

## GIT 环境

使用`git`下载`vsee`平台之前，你需要对`git`做一些环境上的设置：

1. 设置`git config --global core.autocrlf false`，确保空格的字符编码和源码一致。如果你不这么做，将会导致`git`认为大部文件是与服务端不版本不一致的，当你查看差异时，会发现本地的空格采用的是`CRLF`，而服务器文件采用的是`LF`的。
2. 设置`git config core.ignorecase false`，确保本地是能明确识别大小写的。如果你不这么做，一些文件名仅仅是改动了大小写，是不会被侦测到的，有可能导致本地项目无法运行。
3. 设置用户名和邮箱，确保其他开发者可以识别提交者。

## Node & NPM 环境

依据`vsee`现有可运行项目的所在环境设置：

1. NodeJS目前版本`12.14.0`，实际开发不应低于`10.x.x`;
2. NPM目前版本`6.13.4`，实际开发不应低于`6.10`;
